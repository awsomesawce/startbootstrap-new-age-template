So what's the big deal about Sublime Text?

I guess it's one of those text editors that are light and require some setting up

It's a bit faster I think than VSCodium, but it's definitely not as feature packed out of the gate

Microsoft knows what they're doing when it comes to nice text editors

Sublime does have some nice Emmet action going on, and when you type "</" it automatically closes the last open element. Very nice to see.

I actually kind of prefer that instead of the auto=-tag function where it automatically types the end tag out for you

I mean, what if you want the end tag to be in a different location? Say you want to Bolden up a word.

In VSC and other code editors, you'll have to cut and paste the end tag because it types it out for you
